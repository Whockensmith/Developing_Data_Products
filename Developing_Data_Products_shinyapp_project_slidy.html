Overview

The shiny application that I built
MPG vs Number of cylinders, Gears, Transmission Type, and Engine Displacement
allows the user to compare the MPG of cars to the cars number of cylinders, gears, the type of transmission and the cars engine displacement.

The application contains a Bar and Whiskers plot that allows the user to do two dimensional comparisons between MPG and number of cylinders or number of gears or transmission type. The user can also switch to a Coplot that will add a third dimension of comparison by adding engine displacement.

The data is from the mtcars table with in R-Studio. It was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973-74 models).

Using the Application

On the top left choose the type of graph with the Pick Chart Type: dropdown.
Below that is a check box to select or hide outliers. This in only for the Box and Whiskers charts.
Next is a dropdown box for selecting the Factor Variable. This is conveniently called the Select Factor Variable:. The choices are: Cylinders, Transmission, and Gears.


The application is easy to opperate. Reading the graphs are fairly straight forward.

Reproducing The Application

The application was build using Rstudio (version 3.3.1 (2016-06-21)) and knited to Rstudios shinyapp.io cloud server.

The server.R and ui.R files can be found in folder /Dev_data_products.

The .dcf file is in the /rsconnect subfolder.

The Slidify slide show is located in the root directory /Developing_Data_Products.

The shinyapp is located on the shinyoi server at (https://whockensmith.shinyapps.io/developing_data_products_shinyapp_project/)
